Reaquirements :
>Python 3.10

For widows users :
```console 
pip install windows-curses
```

Run Project :
```console
sudo python setup.py install
```
Generate Spring Project :
```console 
sg i name_of_project
cd name_of_project
```

Generate New entity :
```console
sg c entity_name
```

Generate repository :
```console
sg r
```

Generate association :
```console
sg a
```

Generate Enum :
```console
sg e
```

DTOs are generated automaticlly
> for lists you need to add code manually
